Formatieren von Text und Einfügen von einfachen Zeichnungen

In 5Code können Sie formatierten Text sehr gut nutzen, um 

  • sich in Aufgaben gut zu orientieren
  • eigene Anmerkungen in direkt in die Aufgaben zu schreiben
  • Ihre eigenen Überlegungen zur Lösung übersichtlich zu schreiben

Formatieren von Text

Im Folgenden werden die Möglichkeiten zur Formatierung beschrieben.

Alle Formatierungen können über die Tool-Leiste oben im Fenster erreicht werden:

Die Tooltips, d.h. der Text, der erscheint, wenn man die Maus über ein Icon führt, erläutert kurz die Funktion und auch den shortcut, d.h. das Tastatur-Kürzel 

Die Format-Funktionen sind:

  • Absatz-Formatierungen
    • Überschrift 1, 2, 3 (Strg-Shift-1, 2, 3)
    • Aufzählung und Bullet-Liste
    • Ein- und Ausrückung eines Absatzes
  • Einfügen eines Bildes aus einem File (jpg, gif, png) 
  • Buchstaben-Formatierungen
    • Notizen (Strg-N) - sinnvoll, um eigene Notizen im Aufgaben-Text zu markieren, um sie schnell zu finden
    • Textmarker (Strg-1,2,3,4)- um im Aufgabentext wichtige Stellen zu markieren
    • Code - reine Formatierung, nicht ausführbar (-> dafür gibt es ein anderes Feature)
    • Fett (Strg-B), Kursiv (Strg-I), Unterstrichen (Strg-U), Durchgestrichen, ohne Formatierung
Gegebenenfalls sehen Sie noch nicht alle der hier angegebenen Formatierungen.

Hinweise und Tipps 

Manchmal braucht man zusätzlich zur Aufgabenstellung noch ein paar zusätzliche Hinweise. Oder man findet keinen Ansatz zur Lösung und braucht einen Tipp. 

Damit Hinweise und Tipps nicht Platz im Text verbrauchen und damit die Übersicht verringern, werden sie zunächst verdeckt. Man kann sie aber mit einem Klick aufklappen. 

Sie können/sollten also zunächst versuchen, ohne Hinweis oder Tipp auszukommen. Wenn nötig, bekommen Sie aber diese zusätzliche Hilfestellung. 

Um das Nutzen von Tipps nicht allzu einfach zu machen, und um Sie so zum Selberdenken anzuregen, "kosten" Tipps etwas. Die "Währung" in der Sie Tipps bezahlen heißt 5Coin und wird in einem späteren Abschnitt beschrieben (siehe Kapitel Features und 5Coins).

Konkrete Aufgaben

Tun Sie etwas

Die Aufgabentexte bestehen häufig aus viel Text, zur Einführung in die Problemstellung, Hinweise und Tipps (s.o.), Herangehensweise, Abgrenzungen, Ratschlägen, etc. Um da die konkrete Anweisung, was zu tun ist, nicht zu übersehen, werden diese besonders markiert, z.B. so:

Lesen und Verstehen Sie den Absatz oben.

Antworten Sie auf eine Frage

Manchmal ist die konkrete Aufgabe auch so formuliert, dass eine Antwort erforderlich ist, z.B.:

Warum sollte man Code kommentieren? 

In dem Fall klicken Sie auf das Fragezeichen, dann wird eine Zeile für die Antwort eingefügt. 

Diese ist bereits so formatiert, dass Sie ihre eigenen Antworten im Text immer leicht wiederfinden. 

Einfügen von einfachen Zeichnungen

Wenn, in späteren Terminen, Daten und ihre Zusammenhänge oder Abläufe beschrieben werden sollen, geht das häufig am anschaulichsten mit einfachen Zeichnungen. Diese können direkt in einem 5Code-Dokument gezeichnet werden. Dazu fügen Sie eine neue Zeichnung ein. 

  •  Mit dem Shortcut Strg-Shift-G
  • Mit dem Zeichnung-Icon aus der Toolbar oben rechts im Fenster: 

Diese Zeichenfläche bietet einfache Tools an:

  • Linie, Rechteck, Kreis, Freihand
  • Text - der Text kann (noch) nur in der Toolbar editiert werden
  • Raster -> beim Zeichnen oder Bewegen werden Objekte am Raster ausgerichtet
  • Elemente können gelöscht sowie nach oben und unten orientiert werden, um die Verdeckung zu steuern
  • Die Größe der Zeichenfläche kann über die x- und y-Größe eingestellt werden

Alle Icons werden ebenfalls mit einem Tooltip erklärt.
