Einfache Arrays

Wir beginnen auf einfache Weise, den Umgang mit Arrays zu erlernen:

  • Wir erzeugen ein Array.
  • Wir füllen es mit Werten.
  • Wir geben die Werte wieder (auf die Konsole) aus.

Sie können hierbei folgende Skills erlernen

Ein Array erzeugen

Wie gelernt, ist ein Array ein Referenz-Datentyp.

D.h. eine Array-Variable beinhaltet nicht das Array selber, d.h. nicht den Speicherplatz für die einzelnen Elemente, sondern "nur" eine Referenz auf diesen Speicherplatz. Der Speicherplatz für eine feste (!) Anzahl von Elementen des Arrays muss dann explizit zusätzlich erzeugt ("allokiert") werden.

Erzeugen Sie 

  • eine Variable zahlenFeld, die ein Array von int referenzieren kann,
  • ein Array von 5 int-Elementen, das von zahlenFeld referenziert wird.

in einem neuen Codeblock:

int[] zahlenFeld = new int[5];

 

 

Ein Array füllen

Alle Elemente müssen nacheinander explizit gefüllt werden.

Dazu schreiben Sie nun eine Schleife, die das oben definierte und allokierte Array mit aufsteigenden Zahlen von 1 an füllt, d.h. mit 1,2,3, ...

Hinweise zum Umgang mit Arrays

Tun Sie das in einem neuen Codeblock.

for(int i = 0; i < zahlenFeld.length; i++) {
  zahlenFeld[i] = i + 1;
}

 

Skizzieren Sie nun in der Zeichnung unten, wie die Speicherinhalte aussehen. In der Zeichnung sind bereits Textfelder für die Werte im Speicher vorbereitet, Sie müssen nur passende Werte eintragen.

Für die Referenz zeichen Sie einfach Freihand einen Pfeil ein

Ein Array ausgeben

Um zu kontrollieren, dass das oben gefüllte Array auch wirklich mit den richtigen Werten gefüllt ist, geben Sie nun alle Elemente des Arrays auf die Konsole aus - natürlich ebenfalls in einer Schleife, in der sie die o.a. Hinweise ebenfalls beachten.

Tun Sie das in einem neuen Codeblock.

for(int i = 0; i < zahlenFeld.length; i++) {
  System.out.print(zahlenFeld[i] + " ");
}

 

 

Array-Referenzen kopieren

Da ein Array ein Referenz-Datentyp ist, wird bei einer Zuweisung auch nicht das komplette Array mit allen Elementen kopiert, sondern nur die Referenz darauf in eine andere Array-Variable kopiert.

Definieren Sie ein weiteres Array anderesFeld von int, d.h. Sie definieren nur die Variable - allokieren Sie aber keinen Platz für Elemente.

Weisen Sie nun der Variablen anderesFeld die oben definierte Variable zahlenFeld zu.

Schreiben Sie nun einen neuen Wert in ein Element von anderesFeld. Geben Sie dann den Wert des Elements am gleichen Index in zahlenFeld aus. Verifizieren Sie, dass das der gleiche Wert ist.

Tun Sie das in einem neuen Codeblock.

int[] anderesFeld = zahlenFeld;

zahlenFeld[0] = 10;

System.out.println("");
System.out.println("zahlenFeld[0]: " + zahlenFeld[0]);
System.out.println("anderesFeld[0]: " + anderesFeld[0]);
