Geschwindigkeitstest mit einfacher Verzweigung

Entwickeln Sie als einfaches Beispiel einen Geschwindigkeitstest. Dabei soll festgestellt werden, ob der Tachowert die Höchstgeschwindigkeit überschreitet. Für die Unter- und Überschreitung des Tachowertes sollen unterschiedliche Meldungen ausgegeben werden.

Für das aktive Lesen des Texts empfiehlt es sich, die Textmarker zu nutzen.
Falls Sie sie noch nicht freigeschaltet haben, wäre das jetzt ein guter Zeitpunkt

Genügend 5Coins sollten Sie eigentlich haben. 

Mit den Textmarkern können Sie die einzelnen Bestandteile, Variablen, Werte, Abläufe markieren

Suchen Sie sich selber die dazu passenden Farben aus. Vier stehen Ihnen zur Verfügung.

Die Umsetzung machen Sie der Reihe nach in einzelnen Codeblöcken. Testen Sie Ihren Code der Reihe nach wie schon bekannt mit Menü -> Bis hierhin alle Codeblöcke ausführen oder dem Shortcut Strg-Shift-E oder den Buttons am Code-Block. 

a) Definieren Sie zuerst für den Anzeigewert eines Tachometers und für die erlaubte Höchstgeschwindigkeit eines Autos zwei passende Variablen. Geben Sie die gefahrene Geschwindigkeit und die Höchstgeschwindigkeit (auf die Konsole) aus.

Verwenden Sie passende Datentypen und beschreibende Namen ohne Abk. für die Variablen.

Tun Sie das, hier und auch in den folgenden Unteraufgaben, jeweils in einem eigenen Codeblock:

double aktuelleGeschwindigkeit = 49.5;
double erlaubteMaxGeschwindigkeit = 50.0;

System.out.println("Aktuell: " + aktuelleGeschwindigkeit + "; Erlaubt: " + erlaubteMaxGeschwindigkeit);

b) Definieren Sie dann eine Variable, die angibt, ob Sie zu schnell fahren. („Ob“ bedeutet, dass diese Aussage wahr oder falsch ist.).

Den Namen einer boolean-Variablen sollten Sie immer so wählen, dass Sie verständlichen Code erzeugen, z.B. boolean istZuGross oder hatRaeder oder kannFahren.

Vergleichen Sie die gefahrene und die erlaubte Geschwindigkeit und weisen Sie der gerade definierten Variable das Ergebnis zu. Geben Sie dann den Wert dieser Variable aus. Tun Sie das in einem eigenen Codeblock:

double aktuelleGeschwindigkeit = 51.5;
double erlaubteMaxGeschwindigkeit = 50.0;

boolean istZuSchnell = aktuelleGeschwindigkeit > erlaubteMaxGeschwindigkeit;

System.out.println("istZuSchnell: " + istZuSchnell);

c) Dann verwenden Sie eine Verzweigung (Fallunterscheidung), um je nach Ergebnis des Vergleichs (ok/zu schnell) eine passende Text-Meldung auf die Konsole ausgeben.

Es ist dabei nicht notwendig, den Wert der boolean-Variable mit true zu vergleichen
Eine boolean-Variable beinhaltet bereits den Wert der Bedingung, die bei if angegeben wird.

Erkennen Sie, dass es nicht (!) notwendig ist, den Wert dieser Variable mit true oder false zu vergleichen (z.B. mit if(var == true)). Die Angabe der boolean-Variablen if (hatRaeder) hat bereits den Datentyp boolean und ist daher bereits die Bedingung. Diese Form sollten Sie daher verwenden. Der Ausdruck sieht dann richtig und lesbar aus, wenn Sie die Variable richtig benennen (s.o.).

Tun Sie das in einem eigenen Codeblock:

double aktuelleGeschwindigkeit = 51.5;
double erlaubteMaxGeschwindigkeit = 50.0;

boolean istZuSchnell = aktuelleGeschwindigkeit > erlaubteMaxGeschwindigkeit;

if(istZuSchnell) {
  System.out.println("zu schnell");
} else {
  System.out.println("ok");
}

 

Fehlermöglichkeiten

Auch bei so einfachen Kontrollstrukturen wie Verzweigungen kann man etliche Fehler machen. Damit wir diese Fehler kennen, probieren wir die wichtigsten nun aus. Sie können Sie dann erkennen und beheben, wenn sie Ihnen selber unterlaufen. 

Setzen Sie ein „;“ fehlerhafterweise hinter die Klammer nach der Bedingung eines if-statements, z.B. so:

if(bedingung);

    System.out.println(„bedingung ist erfüllt“);

Tun Sie das in einen neuen Codeblock:

if(true); else
  System.out.println("true");

if(false); else
  System.out.println("false");

 

a) Welche Ausgabe erhalten Sie auf der Konsole jeweils für eine true und eine false bedingung?

(Tipp: Sie können hier auch die Ausgabe aus der Konsole hierher kopieren.) 

Sowohl true, als auch false wird ausgegeben

b) Welches Problem ergibt sich also mit diesem fehlerhaften Setzen eines „;“ ? Tragen Sie hier Ihre Antwort ein.

Antwort: Durch das ";" wird die if-Bedinung sofort beendet und der normale Code weiter ausgeführt.

c) Hängen Sie ein else - statement an die oben geschriebene Fallunterscheidung an. Welche Fehlermeldung erhalten Sie?

Antwort: Gar keine

d) Finden Sie noch andere Fehlermöglichkeiten? 

Beliebter Fehler ist folgendes:

 

if(true)
  System.out.println("1");
  System.out.println("2");

Man denkt, dass 1 und 2 nur bei if == true ausgeführt wird, jedoch wird nur 1 ausgeführt, weil keine Block-Schreibweise verwendet wurde.

Verschachtelte Verzweigung

Nun wollen wir nicht nur eine einfache Verzweigung sondern mehrere, verschachtelte Verzweigungen anwenden.

Anforderung:

Wenn die Höchstgeschwindigkeit überschritten wird, gibt es verschiedene Stufen der Sanktionen, die sich nach der Höhe der Differenz ergibt:

  • Wenn die Differenz kleiner oder gleich null ist, gibt es natürlich keine Strafe
  • sonst: Wenn die Differenz kleiner als 10 km/h ist, ist die Strafe 10 Euro
  • sonst: Wenn die Differenz kleiner als 50 km/h ist, ist die Strafe 300 Euro, 1 Strafpunkt
  • sonst: Wenn die Differenz kleiner als 70 km/h ist, ist die Strafe 500 Euro, 3 Strafpunkte und ein Führerscheinentzug auf 1 Jahr
  • Ansonsten ist der Führerschein für immer weg

Geben Sie jeweilige Strafe als Text auf die Konsole aus. Das Muster dafür ist: „Sie sind xxx km/h zu schnell gefahren. Die Strafe dafür ist yyy“.

Lösungsweg:

Sie ermitteln zuerst die Differenz zwischen gefahrener und erlaubter Geschwindigkeit und speichern sie in einer passend benannten Variablen eines passenden Datentyps.

Zur Ermittlung des Texts bauen Sie zuerst einen String passend zusammen. Den geben Sie dann abschließend auf die Konsole aus.

Wie Sie einen String zusammenbauen, haben wir bereits gelernt.

Den Wert des auszugebenden Strings ermitteln Sie dann dadurch, dass Sie Verzweigungen "verschachteln", d.h. in einem Zweig einer Verzweigung eine weitere Verzweigung einbauen:

- Als Bedingung für jede Verzweigung prüfen Sie schrittweise jeweils die Höhe der Differenz:

- Falls die Differenz der ersten Stufe entspricht, bauen Sie den passenden Text zusammen.

- Andernfalls testen Sie, ob die Differenz der zweiten Stufe entspricht und so weiter.

Sie schreiben also Verzweigungen innerhalb von Verzweigungen. Dabei müssen Sie beachten:

  • Folgen Sie der o.a. Reihenfolge 1..5 für das Prüfen der Höhe der Differenz.
  • Rücken Sie den Code bei der Verschachtelung so ein, dass Sie den Überblick behalten können, um den Code verstehen und so formal und logisch korrekt schreiben können.
  • Beschreiben Sie bei jedem Schritt die Anforderung in einem //-Kommentar.
    • Diese können Sie aus der Aufgabe oben in Ihren Code kopieren. 

Den zusammengebauten String geben Sie erst am Schluss, nach der verschachtelten Verzweigung auf die Konsole aus.

Tun Sie das in einem eigenen Codeblock:

double aktuelleGeschwindigkeit = 75.5;
double erlaubteMaxGeschwindigkeit = 50.0;

double differenz = aktuelleGeschwindigkeit - erlaubteMaxGeschwindigkeit;

if(differenz >= 0) {
  if(differenz < 10) {
    System.out.println("Sie sind " + differenz + " km/h zu schnell gefahren. Die Strafe dafür ist 10€");
  } else if (differenz < 50) {
    System.out.println("Sie sind " + differenz + " km/h zu schnell gefahren. Die Strafe dafür ist 300€, 1 Strafpunkt");
  } else if (differenz < 70) {
    System.out.println("Sie sind " + differenz + " km/h zu schnell gefahren. Die Strafe dafür ist 500€, 3 Strafpunkt und Führerscheinentzug für 3 Jahr");
  } else {
    System.out.println("Sie sind " + differenz + " km/h zu schnell gefahren. Die Strafe dafür ist lebenslanger Führerscheinentzug");
  }
} else {
  System.out.println("Alles super!");
}

 

 

Mehrfache Fallunterscheidung

Wenn man nicht nur eine wahr/falsch- bzw. true/false-Bedingung - wie bisher - auswerten möchte, sondern eine mehrfache Fallunterscheidung mit verschiedenen Werten vornehmen möchte, dann kann die Verwendung von mehrfach verschachtelten if-else-Konstrukten unübersichtlich werden.

Daher gibt es auch in Java hierfür die Kontrollstruktur switch-case, die Sie nun einsetzen sollen.

Auswertung von einzelnen Werten

Bei interaktiven Anwendungen werden häufig Eingaben vom Anwender angefordert, je nach Eingabe passieren dann unterschiedliche Aktionen.

Da wir noch nicht mit Eingaben umgehen können, gehen wir wie folgt vor:

  • Definieren Sie eine Variable, die den Buchstaben der Eingabe speichert
    • verwenden Sie einen passenden Datentypen
    • und geben Sie einen beschreibenden Namen
    • wobei Sie die Konventionen der Namensgebung für Variablen in Java beachten
  • Je nach Eingabe geben Sie unterschiedliche Texte auf die Konsole aus.
    • bei einem q oder Q: "Programm wird beendet"
    • bei einem s oder S: "Daten werden gespeichert"
    • bei einem r oder R: "Daten werden berechnet"
    • bei allen anderen Eingaben: "Nur q,Q, s, S, r und R sind erlaubt".

Tun Sie das, natürlich in einem Codeblock:

char eingabe = 'S';

String ausgabe = null;
switch(eingabe) {
    case 'Q':
    case 'q':
      ausgabe = "Programm wird beendet";
        break;
    case 'S':
    case 's':
        ausgabe = "Daten werden gespeichert";
        break;
    case 'R':
    case 'r':
        ausgabe = "Daten werden berechnet";
        break;
    default:
        ausgabe = "Nur q, Q, s, S, r und R sind erlaubt";
}

System.out.println(ausgabe);

 

 

Testen Sie diese Fallunterscheidung mit unterschiedlichen, erlaubten und nicht erlaubten Werten für die Eingabe. Dazu tragen Sie nacheinander unterschiedliche Werte in die Eingabe-Variable ein und führen den Codeblock aus.

Auswertung von Strings

In Java kann man nicht nur eine mehrfache Fallunterscheidung nach einem Zahlen- oder Buchstabenwert treffen, sondern auch nach Werten einer String-Variablen. Das ist nützlich, wenn man je nach Datei-Endung verschiedene Programme aufrufen möchte, um diese Dateien zu bearbeiten.

Bauen Sie in einem neuen Codeblock eine mehrfache Fallunterscheidung auf, die unterschiedliche Texte auf die Konsole ausgibt, je nach der Endung, die natürlich in einer Variable steht.

String name = "test.jpg";

// Bei name = "test.png" erhalt man die Zahl 4, weil an der
// 4. Stellen der letzte Punkte verwendet wird.
int stelleDesLetztenPunktes = name.lastIndexOf('.');

if(stelleDesLetztenPunktes > 0) {
  String type = name.substring(stelleDesLetztenPunktes + 1);
    
    String ausgabe;
    switch(type.toLowerCase()) {
        case "jpg":
      ausgabe = "Bilddatei wird geöffnet";
            break;
        case "doc":
            ausgabe = "Textdatei wird geöffnet";
            break;
        case "xls":
            ausgabe = "Spreadsheetdatei wird geöffnet";
            break;
        default:
            ausgabe = "Nur .jpg, .doc und xls sind erlaubt";
    }
    
    System.out.println(ausgabe);
} else {
  System.out.println("Die Datei hat keine Endung!");
}
  • bei .jpg geben Sie aus "Bilddatei wird geöffnet"
  • bei .doc geben Sie aus "Textdatei wird geöffnet"
  • bei .xls geben Sie aus "Spreadsheetdatei wird geöffnet"
  • bei anderen Endungen geben Sie aus "Nur .jpg, .doc und .xls sind erlaubt"

Testen Sie diese Fallunterscheidung mit unterschiedlichen, erlaubten und nicht erlaubten Werten für die Endung.
