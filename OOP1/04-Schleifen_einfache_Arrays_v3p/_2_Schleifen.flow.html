Schleifen

Wir probieren in dieser Aufgabe die verschiedenen Arten von Schleifen in Java aus.

Dazu verwenden wir zunächst sehr einfache Beispiele, in denen wir nur Reihen von Zahlen auf verschiedene Arten und Weisen erzeugen.

Da die Beispiele sehr einfach sind, können Sie sich auf den reinen Umgang mit Schleifen konzentrieren. In der nächsten Aufgabe wenden wir dann Schleifen auch an.

Einfache Schleifen

Geben Sie in einer Schleife alle Zahlen n von 1 bis 10 (auf die Konsole) aus.

Formulieren Sie diese Schleife als while-Schleife.

Probieren Sie das in einem neuen Code-Block aus.

int zaehler = 1;
while(zaehler <= 10) {
  System.out.println(zaehler);
  zaehler++;
}

 

Andere Schrittweiten

Geben Sie in einer Schleife alle Zahlen n von 20 bis 11 in Schritten von 3 aus.

Formulieren Sie diese Schleife

  • als for-Schleife und
  • als do-while-Schleife.

Probieren Sie das in einem neuen Code-Block aus.

for(int n = 20; n > 10; n -= 3) {
  System.out.println(n);
}
int n = 20;
do {
    System.out.println(n);
    n -= 3;
} while(n > 10);

 

Testen Sie diesen Code mit verschiedenen (kleinen) Werten für n, z.B.: 3, 8, 12.

 

Fehlermöglichkeiten

Nun probieren wir, welche Fehler man beim Schreiben von Schleifen machen kann - damit wir diese Fehler erkennen und beheben können.

Kopieren Sie dazu den Code der vorherigen Aufgabe in einen neuen Codeblock.

for() {
    int n = 20; n > 10; n -= 3
  System.out.println(n);
}

Dann testen wir darin ein paar Fehlermöglichkeiten:

a) Geben Sie testweise die Schleifenvariablen nach Ende des Körpers der for-Schleife (d.h. nach der schließenden Klammer } des Blocks) auf die Konsole aus.

    -> Der Compiler meldet nun einen Fehler, weil die Schleifen-Variable nur im Schleifen-Kopf und im Schleifen-Körper verfügbar ist.

Das ist ein weiteres Feature der Kontrollstruktur "Block"

b) Definieren Sie vor der for-Schleife jeweils eine Variable, die genauso heißt wie die Schleifenvariable. Warum meldet der Compiler nun einen Fehler?

Probieren Sie das ebenfalls auf der oben erstellen Code-Kopie aus.

 

int n = 12;
for(int n = 20; n > 10; n -= 3) {
  System.out.println(n);
}

Antwort: Es kann nicht mehrmals eine Variable mit dem selben Namen erstellt werden. 

Verschachtelte Schleife

Geben Sie das Einmaleins von 1 bis n als Tabelle auf die Konsole aus.

Beispiel für das Einmaleins von 1 bis 3:

1    2    3
2    4    6
3    6    9

Dafür müssen Sie zwei Schleifen verwenden
Ordnen Sie die Zahlen möglichst geordnet untereinander an

Beschreiben Sie zuächst die beiden Schleifen und codieren Sie sie dann in einem neuen Codeblock

int n = 10;
for(int zeile = 1; zeile <= n; zeile++) {
  for(int spalte = 1; spalte <= n; spalte++) {
      int rechnung = zeile * spalte;
        System.out.print(rechnung + " ");
    }
    System.out.println("");
}

Testen Sie diesen Code dann mit kleinen Werten von n (3,6,9).

Einmaleins mit Überschriften

Erweitern Sie die Ausgabe des Einmaleins so, dass zusätzlich Überschriften für Zeilen und Spalten ausgegeben werden, z.B. so:

     1    2    3

1    1    2    3
2    2    4    6
3    3    6    9

Kopieren Sie dafür den Code von oben und erweitern ihn in einem neuen Codeblock

int n = 5;
for(int zeile = 0; zeile <= n; zeile++) {
  for(int spalte = 0; spalte <= n; spalte++) {
    if(zeile == 0 && spalte == 0) {
        // Links oben in der Ecke soll ein leeres Feld ausgegeben werden
      System.out.print("\t");
    } else {
        if(zeile == 0) {
          // In der ersten Zeile sollen einfach nur die Zahlen 1 - 10 ausgeben werden.
          System.out.print(spalte + "\t");
        } else if(spalte == 0) {
            // Am Anfang einer Zeile soll die Nr. der Zeile ausgegben werden. Dies ist
            // in diesem Fall ein i.
            System.out.print(zeile + "\t");
        } else {
            int rechnung = zeile * spalte;
          System.out.print(rechnung + "\t");
        }
    }  
  }
  System.out.println("");
}
