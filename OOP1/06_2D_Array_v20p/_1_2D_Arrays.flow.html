Variable für Kalender 

Der Adventskalender soll in 4 Zeilen zu je 6 Spalten von Zahlen im Programm abgebildet werden. 

Definieren Sie eine passende Variable für den Adventskalender als lokale Variable in main() und initialisieren Sie diese Variable direkt mit einem passenden 2D-Array-Literal.

Füllen mit ungeprüften Zufallszahlen

Zunächst füllen Sie einen Kalender mit ungeprüften Zufallszahlen. Der Kalender wird dabei als Parameter übergeben. Für die Generierung einer Zufallszahl rufen Sie die vorgegebene Prozedur auf. (Klicken Sie auf den Pfeil, um zum Code der Prozedur zu gelangen).

Definieren Sie eine Prozedur, die alle Elemente im Kalender mit Zufallszahlen von 1 bis 24 füllt.

Zum Testen brauchen wir ein "frisches" Array, dessen Variablen mit 0 initialisiert sind. Definieren Sie daher in main() eine weitere 2D-Array-Variable, der Sie ein neu erzeugtes Array der passenden Größe zuweisen. 

Testen Sie diese Prozedur, indem Sie sie in main() mit der neuen Array-Variable als Parameter aufrufen und danach den Kalender anzeigen. 

Alle Ihre hier erstellten Prozeduren und Funktionen werden durch Aufruf in main() gestestet.

Ist die Zahl bereits im Kalender vorhanden?

Da im Kalender jede Zahl nur 1x vorkommen soll, müssen wir vor dem Setzen prüfen, ob diese Zahl bereits im Kalender vorhanden ist. 

Schreiben Sie eine Funktion, die ermittelt, ob eine als Parameter angegebene Zahl bereits im Kalender vorhanden ist. (Den Kalender übergeben Sie natürlich ebenfalls als Parameter).

Wie finden Sie heraus, ob die Zahl bereits im Kalender steht?

Testen Sie diese Prozedur, indem Sie sie mit verschiedenen Zahlen aufrufen und das Ergebnis anzeigen.

Türen öffnen

Nachdem wir den Kalender nun erfolgreich erstellt und angezeigt haben, wollen wir natürlich auch seine Türen öffnen. 

Wir öffnen die Tür, die eine bestimmte Zahl zeigt. Die geöffnete Türe wird im Kalender dann mit dem Wert 0 dargestellt.

Schreiben Sie eine Prozedur, die eine Türe mit einer bestimmten Zahl öffnet.

Der fertige Adventskalender

Nun haben Sie alle Prozeduren und Funktionen definiert, um einen interaktiven Adventskalender zu bauen.

Jetzt definieren Sie eine Prozedur, die einen bereits mit Zufallszahlen gefüllten Kalender übergeben bekommt, Zahlen von der Konsole einliest und in der richtigen Reihenfolge die Türen öffnet.

Diese Prozedur bauen Sie auf, wie im folgenden beschrieben.

Richtige Reihenfolge einhalten

Um die richtige Reihenfolge einzuhalten, müssen Sie sich merken, welches die nächste Zahl ist, die "dran" ist. Dazu verwenden sie in dieser Prozedur eine passende lokale Variable. Die definieren Sie dort nun. 

Lokale Variable definieren

Ablauf

Die Prozedur soll wie folgt ablaufen: Der Kalender wird angezeigt. Nun liest man eine Zahl von der Konsole ein. Wenn die Zahl der Tür entspricht, die dran ist, wird diese Tür geöffnet, der Kalender wird angezeigt und die nächste Tür ist dran. Wenn das nicht die richtige Zahl war, geben Sie aus, dass diese Zahl noch nicht dran ist. Das wird solange durchlaufen, bis alle Türen geöffnet sind.

Für die Codierung des Ablaufs verwenden wir natürlich die bisher definierten Prozeduren.
