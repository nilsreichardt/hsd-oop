Optional: Grafischer Adventskalender

 -------------------

Diese Aufgabe ist komplett optional.

Bearbeiten Sie ihn aber gerne, um weiter zu üben (immer gut!) und um schon mal einen Ausblick auf eine einfache grafische Ausgabemöglichkeit zu bekommen.

 ------------------

Nachdem der Adventskalender auf der Textkonsole nun funktioniert, wollen wir diesen so ergänzen, dass dieser auch 

  • grafisch dargestellt wird
  • man die Türen mit der Maus öffnen kann.

Die Grundlagen für die dafür verwendete Grafische Konsole finden Sie in diesem Dokument: Grafische Konsole. Bei der Bearbeitung der folgenden Aufgaben finden Sie dort eine kurze Beschreibung, wie Sie damit arbeiten können. 

Wenn Sie einem Link, wie dem zur Beschreibung der grafischen Konsole, folgen, kommen Sie ganz einfach hierher wieder zurück zu diesem Dokument.

Grafik-Konsole initialisieren

Damit die neuen grafischen Möglichkeiten genutzt werden können, muss die Grafik-Konsole bereit gemacht, d.h. initialisiert werden. Das geschieht mit dem Aufruf GrafikKonsole.clearGrafik().

Um die im Folgenden erstellten Prozeduren testen zu können, fügen Sie daher diesen Aufruf in main() ein.

 

Auch grafische Türen in einem Array speichern

Bisher haben wir ein Array von Zahlen verwendet, um die Türen zu speichern. Nun wird jede Tür wird außerdem grafisch dargestellt. Das geht ganz einfach mit einem Text-Grafikobjekt. Sie nutzen dafür den neuen Datentyp Text, der zum Bereich der Grafik-Konsole gehört. Das ist, wie Array und String auch, ein Referenzdatentyp.

Diese Türen müssen natürlich ebenfalls gespeichert werden, um mit ihnen im Programm arbeiten zu können. Zum Speichern wird ebenfalls ein 2D-Array verwendet, diesmal als 2D-Array von Text.

Eine entsprechende Variable definieren wir später ebenfalls als lokale Variable in der Prozedur, in der der gesamte Ablauf gesteuert wird.

Zum Testen der neuen Prozeduren, definieren Sie nun eine solche Variable in main(), die auch sofort mit einen Array der passenden Größe initialisiert wird

("initialisieren" heißt hier: Es wird nur ein 2D-Array der richtigen Größe erzeugt. Dieses Array wird hier noch nicht mit Werten besetzt, das machen wir im nächsten Schritt).

Türen auch grafisch anzeigen

Die Türen auch grafisch anzuzeigen ist einfach: Bei der Definition eines Text-Grafikobjekts wird es bereits auf der Konsole angezeigt. Da Sie sie gerade schon erzeugt (und im Array gespeichert) haben, brauchen Sie hier nichts ergänzen.

Noch schönere Darstellung

Wenn alles läuft, können Sie optional gerne versuchen, die Darstellung grafisch - mit den zur Verfügung stehenden Mitteln - zu verschönern.
