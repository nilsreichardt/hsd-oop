Grafik-Konsole

Dieses Dokument enthält keine Aufgaben, sondern beschreibt in Kürze die Eigenschaften des neuen Features Grafik-Konsole.

Die Grafik-Konsole teilt sich ihren Platz mit der bisher bekannten Text-Konsole:

Für die Grafikobjekte werden einige spezielle Klassen verwendet: Text, Rechteck, Linie und Kreis. Diese Klassen sind in 5Code vordefiniert.

Ein Grafikobjekt wird mit einem Constructor erzeugt. Die Constructoren der Grafik-Klassen brauchen einige Parameter für die Darstellung des Objekts.

Außerdem gibt es einige Methoden, die ein Grafik-Objekt verändern können.

Initialisierung und Löschen der Grafik-Konsole

Zu Beginn des Nutzung muss die Grafik-Konsole einmal initialisiert werden. Der Code dazu muss also vor der ersten Verwendung von Grafik-Objekten aufgerufen werden.


GrafikKonsole.clearGrafik();

Außerdem kann dieses Kommando dazu genutzt werden, alle Elemente in der Grafik-Konsole zu löschen. 

Farben

Farben werden in Grafik-Objekten als String gespeichert. Für eine Farbe werden Zahlenwerte für Rot, Grün und Blau als Hexadezimalwert "#RRGGBB" angegeben, wie sie auch in der Web-Programmierung verwendet werden (siehe z.B. Colorpicker) . Beispiele für Farben, wie sie in 5Crush verwendet werden:

// schwarz  "#000000"
// weiss     "#FFFFFF"
// rot      "#FF0000"
// gruen     "#00FF00"
// blau     "#0000FF"
// gelb     "#FFFF00"
// magenta  "#FF00FF"
// cyan     "#00FFFF"

Text

Mit dem folgenden Constructor wird ein Text-Objekt erzeugt

  • an der angegebenen Position
  • mit dem darzustellenden Text
  • in der angegebenen Farbe und Größe.
/** erzeugt ein Text-Grafikobjekt und stellt es auf der Grafik-Konsole dar
@param x  x-Koordinate der oberen linken Ecke des Grafikobjekts, 0 ist dabei ganz links
@param y  y-Koordinate der oberen linken Ecke des Grafikobjekts, 0 ist dabei ganz oben
@param text dieser Text wird angezeigt
@param farbe die Farbe des Texts, erlaubte Werte siehe in allgemeiner Beschreibung
@param s Schriftgröße des Texts in Pixeln.
@param reaktion der String, der eingelesen wird, wenn auf das Grafikobjekt geklickt wird
*/
    
Text grafikText =
        new Text(70, 20, "Text"+1, "#0000FF", 20, "1");

Text-Objekte ändern

Objekt auf eine andere Position bewegen

/* Ein Grafik-Objekt kann auf eine andere absolute Position bewegt werden
* @param x x-Koordinate der oberen linken Ecke des Grafikobjekts, 0 ist dabei ganz links
* @param y y-Koordinate der oberen linken Ecke des Grafikobjekts, 0 ist dabei ganz oben
*/
grafikText.moveTo(100, 60);


/* Ein Grafik-Objekt kann relativ um eine Entfernung bewegt werden
* @param dx Entfernung in x, positive Werte bewegen nach rechts
* @param dy Entfernung in y, positive Werte bewegen nach unten
*/
grafikText.moveBy(150, 50);

Farbe ändern

/* Die Farbe kann einzeln geändert werden
* @param farbe die Farbe, erlaubte Werte siehe in allgemeiner Beschreibung
*/
grafikText.setFillcolor("#FF00FF");

Text ändern

 

/* Der Text kann einzeln geändert werden
* @param _text wird neu als Text angezeigt.
*/
grafikText.setText("Neu");

Löschen eines Grafikobjekts

Jedes Grafikobjekt kann gelöscht werden mit dem folgenden Ausdruck:

// erzeugen
Linie weg =
        new Linie(40, 50, 100, 100, "#FF00FF", 3, "4");
// löschen
weg.delete();

 

Interaktion mit Mausklick

Auf jedes Grafikobjekt kann man mit der Maus klicken. 

Wenn das geschieht, kann man von der Konsole einen String einlesen. Dieser String ist der Parameter "reaktion", der beim Erzeugen eines Grafikobjekts angegeben wird.

Wie man von der Konsole einen String einliest, wurde bereits an anderer Stelle ausführlicher dargestellt. Hier reicht der Hinweis auf das Beispiel unten für eine einfache Anwendung.

Um diese Interaktion zu testen,

  • führen Sie einfach den folgenden und alle vorherigen Code-Blöcke aus.
  • Dann werden in der Grafik-Konsole die oben erzeugten Grafik-Objekte angezeigt.
  • Klicken Sie auf eines der Grafik-Objekte, dann wird in der Text-Konsole ein Reaktions-String ausgegeben:
    • Auf den gelben Kreis klicken.
    • -> Ausgabe:
/// Klicken Sie auf ein Grafik-Objekt!
    
String reaktion = Konsole.eingabeString();

System.out.println("Reaktion = " + reaktion);

Wenn man jedem Grafikobjekt, auf dessen Klick man reagieren möchte, einen eigenen String mitgibt, kann man es über diesen String identifizieren und im Spielverlauf eine entsprechende Reaktion auslösen.
